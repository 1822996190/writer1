#智能写作反馈平台
###功能介绍：
    智能写作反馈功能：调用AcaWriter对学术写作进行分析产生反馈 
    图表功能：HTTP请求次数饼状图、HTTP请求信息数据表
    待开发模块：
    测试：CSS动画
###数据库连接
    db.propertes保存了mysql的配置信息，运行前请修改用户名和密码，并建立所需的数据库和表，建表语句可以从writer1.sql中获得。
